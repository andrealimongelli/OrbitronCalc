# **OrbitronCalc** 

## **Description**
OrbitronCalc is a  web-based calculator.

## **Manual**
**AC**  All clear

**CE**  Clear last entry

**M+**  Memory plus

**M-**  Memory minus

**MR**  Memory recall

**MC**  Memory clear

**%**  Ex. 100 % 5 = 5

**√**  Ex. 25 √ = 5

**x^y**  Ex. 2 x^y 3 = 8

**n!**  Ex. 5 n! = 120

**MOD**  Ex. 3 MOD 2 = 1

## **Installation**
You need no installation, just open you preferred browser and start to use OrbitronCalc!

## **Dependencies**
- Font https://fonts.googleapis.com/css2?family=Orbitron&display=swap
